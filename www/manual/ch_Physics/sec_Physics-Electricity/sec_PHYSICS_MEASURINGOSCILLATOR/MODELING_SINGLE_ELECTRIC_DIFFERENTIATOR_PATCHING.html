Patching current

When clamping, a current is directly introduced to the neuron’s body. In the case of a constant current where I=d⁢Qd⁢t, the voltage increase d⁢V on the capacity C of the membrane is d⁢QC=I*d⁢tC, so

dd⁢t⁢V=IC

The direct constant current input dd⁢t⁢VP⁢A⁢T⁢C⁢H to the neuron cell body is a simple constant current that causes a constant membrane’s voltage derivative contribution. However, the currents are not necessarily constant. If the current follows a math function, the derivative of that function should be used. One must be careful that (step-like) sudden changes may produce very steep spikes (see the wave forms in Table 2.1 on differentiating a square wave function); furthermore, as we discuss in section 3.4.5, the step-like concentration change causes exactly the same change in the output voltage, only the time scale differs in a factor of 106.
