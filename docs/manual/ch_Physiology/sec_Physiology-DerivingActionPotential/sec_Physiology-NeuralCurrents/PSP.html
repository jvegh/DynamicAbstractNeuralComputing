Synaptic and rushed-in current

In the case of those currents, as we discussed in the cases of membrane and axon, a saturation-type function multiplied by a decay-type function describes the current, so the voltage derivative is

d⁢Vad⁢t=1α*e⁢x⁢p⁢(-1α*t-1β*t)-1β*e⁢x⁢p⁢(-1β*t)*e⁢x⁢p⁢(1-e⁢x⁢p⁢(-1α*t)) (3.6)

The same voltage derivate (with different parameters a and b) is valid for dd⁢t⁢VM⁢(t) due to the membrane rush-in current (as discussed above, the voltage derivate is proportional to the current through a factor 1/CM). See also Figure 3.10.
