AIS current

The AIS represents a non-distributed resistance RM, and the current flowing through it is

IA⁢I⁢S=-VM-Vr⁢e⁢s⁢tRM (2.19)

(it is an outward current, so it is negative), and its voltage time derivative is

dd⁢t⁢VA⁢I⁢S⁢(t)=-VM⁢(t)-Vr⁢e⁢s⁢tCM⁢RM (2.20)

Notice that this current depends on CM⁢RM, all others on CM. (This current was mis-identified by HH as ’leaking current’: if no other current/voltage derivative is present, the membrane discharges. In resting state the derivative is zero: the condenser is charged up and no leaking current flows.
